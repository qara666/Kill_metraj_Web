# Как задеплоить изменения

## 🚀 Быстрый деплой

### 1. Проверка исправлений
```bash
# Запустить тестовый сервер
cd backend
node simple_server.js

# В новом терминале протестировать
cd ..
node test_final_excel.js
```

### 2. Запуск фронтенда
```bash
cd frontend
npm start
```

### 3. Запуск бекенда
```bash
cd backend
node simple_server.js
```
