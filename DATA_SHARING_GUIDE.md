# 🔗 Руководство по обмену данными

## Обзор системы

Система обмена данными позволяет команде работать над одними и теми же маршрутами и заказами, синхронизируя изменения в реальном времени через URL-ссылки.

## 🚀 Как использовать

### 1. Поделиться данными

1. **Загрузите Excel файл** и создайте маршруты
2. **Нажмите кнопку "Поделиться"** в верхней панели
3. **Скопируйте ссылку** из модального окна
4. **Отправьте ссылку коллегам** через мессенджер, email или любой другой способ

### 2. Импортировать данные

1. **Получите ссылку** от коллеги
2. **Нажмите кнопку "Импорт"** в верхней панели
3. **Вставьте ссылку** в поле ввода
4. **Нажмите "Импортировать"** - все данные загрузятся автоматически

### 3. Автоматическая синхронизация

- **Изменения синхронизируются автоматически** каждые 5 секунд
- **Статус синхронизации** отображается в верхней панели
- **Все участники видят изменения** в реальном времени

## 🔧 Технические детали

### Безопасность данных

- ✅ **Сжатие данных** - используется LZ-String для уменьшения размера URL
- ✅ **Валидация данных** - проверка структуры перед импортом
- ✅ **Временные метки** - отслеживание последних изменений
- ✅ **Версионирование** - поддержка разных версий данных

### Ограничения

- 📏 **Максимальная длина URL** - ограничена браузером (~2000 символов)
- 🔄 **Синхронизация** - работает только в рамках одного браузера/устройства
- 💾 **Хранение** - данные сохраняются в localStorage браузера

### Поддерживаемые данные

- 📊 **Excel данные** - заказы, курьеры, методы оплаты
- 🗺️ **Маршруты** - созданные маршруты с расчетами расстояний
- ⚙️ **Настройки** - конфигурация приложения

## 🎯 Сценарии использования

### Сценарий 1: Командная работа над маршрутами

1. **Менеджер** загружает Excel файл и создает базовые маршруты
2. **Менеджер** генерирует ссылку и отправляет **логисту**
3. **Логист** импортирует данные и оптимизирует маршруты
4. **Логист** генерирует новую ссылку и отправляет **курьерам**
5. **Курьеры** импортируют финальные маршруты

### Сценарий 2: Синхронизация между устройствами

1. **Пользователь** работает на компьютере в офисе
2. **Пользователь** генерирует ссылку и отправляет себе на телефон
3. **Пользователь** импортирует данные на мобильном устройстве
4. **Изменения синхронизируются** между устройствами

### Сценарий 3: Резервное копирование

1. **Пользователь** создает важные маршруты
2. **Пользователь** генерирует ссылку и сохраняет её
3. **При необходимости** пользователь может восстановить данные из ссылки

## 🛠️ Устранение неполадок

### Проблема: Ссылка не работает

**Решение:**
- Проверьте, что ссылка скопирована полностью
- Убедитесь, что в ссылке есть параметр `shared_data`
- Попробуйте открыть ссылку в новом окне браузера

### Проблема: Данные не импортируются

**Решение:**
- Проверьте, что в Excel файле есть данные
- Убедитесь, что созданы маршруты
- Попробуйте очистить localStorage и повторить импорт

### Проблема: Синхронизация не работает

**Решение:**
- Проверьте подключение к интернету
- Убедитесь, что синхронизация включена
- Попробуйте обновить страницу
