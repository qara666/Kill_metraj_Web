# Kill Metraj Web
## Основные разделы

### Панель управления (Dashboard)
*   **Мониторинг**: Отображение актуальных заказов и местоположения курьеров на карте.
*   **Статусы**: Контроль выполнения доставок в реальном времени.

### Автопланировщик (AutoPlanner)
*   **Распределение**: Алгоритмическое назначение заказов на курьеров с учетом нагрузки и маршрутов.
*   **Оптимизация**: Расчет кратчайших путей для снижения времени доставки.

### Работа с курьерами (Couriers & Financials)
*   **База данных**: Учет личных данных и параметров транспорта курьеров.
*   **Финансы**: Расчет выплат, учет штрафов и бонусов на основе выполненных заказов.

### Интеграция с Telegram
*   **Парсинг**: Автоматическое извлечение данных о заказах из входящих сообщений в Telegram-группах.
*   **Бот**: Передача полетных листов курьерам и получение отчетов о доставке.

### Геолокация и Кэширование (v3.0)
*   **Геокодирование**: Преобразование текстовых адресов в координаты через Google Maps.
*   **Многоуровневый кэш**: 
    *   **L1 (Browser)**: Хранение частых адресов в памяти браузера (LRU, 500 записей).
    *   **L2 (Database)**: Общая база гео-данных в PostgreSQL для всех пользователей.
*   **Нормализация**: Автоматическое исправление опечаток и сокращений в адресах для точного поиска.

### Импорт и Экспорт
*   **Excel**: Пакетная загрузка заказов из таблиц и выгрузка отчетов.

## Технический стек
*   **Frontend**: React, TypeScript, Vite, Tailwind CSS.
*   **Backend**: Node.js, Express, Sequelize (ORM).
*   **Database**: PostgreSQL.
*   **API**: Google Maps, Telegram (Telegraf).
