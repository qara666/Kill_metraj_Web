# 🚀 Kill_metraj - Система управления маршрутами

## 📋 Обзор

Kill_metraj - это современная веб-система для управления маршрутами курьеров с поддержкой совместной работы и синхронизации данных в реальном времени.

## ✨ Основные возможности

### 🗺️ Управление маршрутами
- **Создание маршрутов** для курьеров с автоматическим расчетом расстояний
- **Интеграция с Google Maps** для точного расчета маршрутов
- **Оптимизация маршрутов** с учетом пробок и времени
- **Визуализация маршрутов** на интерактивной карте

### 📊 Обработка данных
- **Загрузка Excel файлов** с заказами и курьерами
- **Автоматическая обработка** и валидация данных
- **Геокодирование адресов** для точного позиционирования
- **Статистика и аналитика** по маршрутам и заказам

### 🔗 Совместная работа
- **Обмен данными через URL** - отправляйте ссылки коллегам
- **Синхронизация в реальном времени** - изменения видны всем участникам
- **Кроссбраузерность** - работает во всех современных браузерах
- **Автоматическое сохранение** в localStorage

### 🎨 Современный интерфейс
- **Темная/светлая тема** с автоматическим переключением
- **Адаптивный дизайн** для всех устройств
- **Интуитивный интерфейс** с подсказками и уведомлениями
- **Быстрая навигация** между разделами

## 🚀 Быстрый старт

### 1. Установка зависимостей
```bash
cd frontend
npm install
```

### 2. Настройка Google Maps API
1. Получите API ключ в [Google Cloud Console](https://console.cloud.google.com/)
2. Включите следующие API:
   - Maps JavaScript API
   - Directions API
   - Geocoding API
3. Добавьте ключ в настройки приложения

### 3. Запуск приложения
```bash
npm run dev
```

Приложение будет доступно по адресу: `http://localhost:5173`

## 📖 Руководство пользователя

### Загрузка данных
1. **Перейдите в раздел "Панель управления"**
2. **Нажмите "Выбрать файл"** и загрузите Excel файл
3. **Нажмите "Обработать файл"** для анализа данных
4. **Проверьте результаты** в разделе "Результаты обработки"

### Создание маршрутов
1. **Перейдите в раздел "Маршруты"**
2. **Выберите курьера** из списка
3. **Выберите заказы** для маршрута
4. **Нажмите "Создать маршрут"**
5. **Дождитесь расчета расстояния** через Google Maps

### Обмен данными
1. **Нажмите кнопку "Поделиться"** в верхней панели
2. **Скопируйте ссылку** из модального окна
3. **Отправьте ссылку коллегам**
4. **Коллеги нажимают "Импорт"** и вставляют ссылку

## 🔧 Технические детали

### Архитектура
- **Frontend**: React 18 + TypeScript + Vite
- **Стилизация**: Tailwind CSS
- **Состояние**: React Context + localStorage
- **Карты**: Google Maps JavaScript API
- **Сжатие данных**: LZ-String для URL

### Структура проекта
```
frontend/
├── src/
│   ├── components/          # React компоненты
│   ├── contexts/           # React контексты
│   ├── hooks/              # Пользовательские хуки
│   ├── pages/              # Страницы приложения
│   ├── services/           # API сервисы
│   ├── utils/              # Утилиты
│   └── types/              # TypeScript типы
├── public/                 # Статические файлы
└── package.json           # Зависимости
```

### Ключевые компоненты
- **Layout** - основной макет с навигацией
- **RouteManagement** - управление маршрутами
- **DataSharing** - обмен данными
- **ExcelUploadSection** - загрузка Excel файлов
- **SyncStatus** - статус синхронизации

## 🐛 Устранение неполадок

### Частые проблемы

#### Google Maps не загружается
- Проверьте API ключ в настройках
- Убедитесь, что включены необходимые API
- Проверьте квоты и лимиты

#### Данные не сохраняются
- Проверьте доступность localStorage
- Убедитесь, что браузер поддерживает localStorage
- Попробуйте очистить кэш браузера

#### Синхронизация не работает
- Проверьте подключение к интернету
- Убедитесь, что синхронизация включена
- Попробуйте обновить страницу
