# Документация по компоненту CourierCard

## Общая информация
Компонент `CourierCard` предназначен для отображения информации о курьере в визуальном формате. Он выводит важные данные о курьере, такие как статус, информация о доставках и прочие детали, а также предоставляет возможность для взаимодействия с пользователем.

## Функции
Каждая функция адаптирована к различным сценариям и обеспечивает функциональность, связанную с управлением курьерами.

### Функции компонента
1. **formatDistance**: 
   - Описание: Форматирует расстояние для отображения.
   - Параметры: 
     - `distance`: Расстояние, которое нужно отформатировать.
   - Рекомендации: Необходимо добавить и проверить типизацию Props.

2. **getVehicleIcon**: 
   - Описание: Возвращает иконку транспортного средства на основе типа.
   - Параметры:
     - `vehicleType`: Тип транспортного средства.
   - Рекомендации: Необходимо добавить и проверить типизацию Props.

3. **getStatusColor**: 
   - Описание: Возвращает цвет статуса курьера.
   - Параметры:
     - `status`: Текущий статус курьера.
   - Рекомендации: Необходимо добавить и проверить типизацию Props.

## Рекомендации по улучшению
- Разделить компонент на более мелкие функции для повышения читаемости и сопровождения.
- Убедиться в наличии и корректности типизации Props для всех используемых параметров.
