# Новые функции и проблемы проекта


## 2. AutoPlanner
### Проблемы:
- Сложности с кластеризацией заказов, что затрудняет оптимизацию маршрутов.

### Предложения по улучшению:
- Внедрить алгоритмы кластеризации для группировки заказов по близости и оптимизации маршрутов.
- Добавить возможность ручной донастройки маршрутов.
- Реализовать автоматизированную систему назначения курьеров на основе их доступности и производительности.

- Интегрировать геолокационные системы для оптимизации отслеживания по геофенсам.

### Предложения по улучшению:
- Реализовать графическую визуализацию с данными от Swagger API для удобного анализа.
- Внедрить динамический выбор типов визуализации для анализа.
