## Как протестировать

### 1. Откройте консоль браузера (F12)
Перейдите на вкладку Console

### 2. Загрузите Excel файл
Используйте ручную загрузку через интерфейс

### 3. Проверьте логи
Вы должны увидеть:
```
📋 [Excel Processor] Первые 5 строк файла для поиска заголовков:
✅ [Excel Processor] Найдена строка заголовков в строке X
📋 Заголовки Excel (все): [...]
📋 Всего строк данных: X
```

### 4. Если заказы НЕ найдены (0 orders)
Проверьте вывод:
```
⚠️ [Excel Processor] НЕ НАЙДЕНО ЗАКАЗОВ! Проверьте:
  1. Заголовки файла: [...]
  2. Первая строка данных: [...]
  3. Найденные колонки адресов: [...]
  4. Всего строк данных: X
  5. Ошибки: [...]
```

### 5. Что проверить в логах

#### Заголовки должны содержать:
- Колонку с адресом (адрес, address, куда, доставка, и т.д.)
- Номер заказа (№, номер, number) ИЛИ
- Сумму (сумма, amount, цена)

#### Строки данных должны:
- Содержать адрес с номером дома
- Иметь хотя бы 3 символа в адресе
- Содержать буквы (не только цифры)

### 6. Примеры ВАЛИДНЫХ адресов
✅ "вул Хрещатик 5"
✅ "Ленина 10"
✅ "проспект Перемоги 25"
✅ "Київ, вул. Шевченка 12"
✅ "ул Пушкина 7а"

### 7. Примеры НЕВАЛИДНЫХ адресов
❌ "12" (только номер)
❌ "+380501234567" (телефон)
❌ "Іван Петров" (имя)
❌ "Безготівка" (способ оплаты)
❌ "Зона 3" (зона доставки)
