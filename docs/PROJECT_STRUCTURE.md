# Структура проекта Kill_metraj_Web

## Общая информация
Проект Kill_metraj_Web состоит из двух основных частей: **Backend** и **Frontend**. 

### Backend
Backend проекта отвечает за обработку логики и данных. Главные компоненты включают:
- **Контроллеры**: управляют обработкой запросов и ответов. 
  - UploadController.js: Загрузка файлов.
  - RouteController.js: Управление маршрутами.
- **Сервисы**: содержат бизнес-логику и интеграцию с API.
  - TelegramService.js: Взаимодействие с Telegram API.
  - ExcelService.js: Обработка Excel файлов.
- **Модели**: представляют данные и их структуру.
  - Courier.js: Модель курьера.
- **Middleware**: промежуточное ПО для обработки запросов.
  - rateLimiter.js: Ограничение частоты запросов.

### Frontend
Frontend проекта отвечает за интерфейс пользователя и его взаимодействие с бэкендом. Главные компоненты включают:
- **Страницы**: основные временные страницы.
  - Dashboard.tsx: Централизованный мониторинг.
  - AutoPlanner.tsx: Авто-планировщик.
- **Компоненты**: переиспользуемые интерфейсные элементы.
  - AIFeatures.tsx: Функции ИИ.
  - ExcelUploadSection.tsx: Секция загрузки Excel файлов.
- **Утилиты**: вспомогательные функции для работы компонентов.
  - dataSharing.ts: Функции обмена данными.
