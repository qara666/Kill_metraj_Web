# Обновление ExcelService для новой структуры таблицы

## 🎯 Проблема
Пользователь показал новую структуру таблицы с множественными колонками и подколонками, которую нужно было распознавать.

## ✅ Решение

### 1. Обновлено распознавание заголовков
Добавлены все новые поля из структуры таблицы:

#### Основные поля:
- **Номер** - номер заказа
- **Состояние** - статус заказа (Исполнен, Новый, и т.д.)
- **Тип заказа** - Доставка/Самовивіз
- **Телефон** - контактный телефон

#### Информация о заказчике:
- **Заказчик** - имя клиента
- **Всего заказов** - количество заказов клиента
- **Комментарий к заказчику** - заметки о клиенте

#### Адресная информация:
- **Адрес** - полный адрес доставки
- **Комментарий к адресу** - инструкции по адресу
- **Зона доставки** - зона доставки (Зона 0, Зона 1, и т.д.)
- **Время доставки** - время в минутах

#### Временные метки:
- **Дата создания** - когда создан заказ
- **Время на кухню** - время передачи на кухню
- **Доставить к** - время доставки
- **Плановое время** - запланированное время

#### Дополнительная информация:
- **Комментарий к заказу** - общие комментарии
- **Общее время** - общее время обработки
- **Скидка %** - процент скидки
- **К оплате** - сумма к оплате
- **Сдача** - сумма сдачи
- **Способ оплаты** - метод оплаты
- **Курьер** - назначенный курьер

- **Номер**: "Номер", "№", "Number", "Order #"

- **Тип заказа**: "Тип заказа", "Тип", "Order Type", "Type"
