# Диагностика проблемы с 0 заказами

## 🔍 Проблема
При обработке Excel файлов система показывает 0 заказов, хотя логика работает правильно.

### 2. Улучшено распознавание заголовков
Добавлены варианты названий колонок:
- **Адреса**: "Адрес", "Адреса", "Address", "Location", "Место", "Адрес доставки"
- **Имена**: "Имя", "Имена", "Name", "Names", "Клиент", "Client", "Заказчик"

### 3. Добавлена детальная диагностика
Система теперь показывает:
- Есть ли колонка с адресами
- Есть ли данные в колонке адресов
- Точную причину почему заказы не создаются

## 🎯 Возможные причины 0 заказов

### 1. Нет колонки с адресами
**Симптомы**: Ошибка "Нет колонки с адресами"
**Решение**: Добавьте колонку с названием "Адрес", "Адреса", "Address" или "Location"

### 2. Колонка адресов есть, но все ячейки пустые
**Симптомы**: Предупреждение "Заказы не созданы. Проверьте, что в колонке адресов есть данные"
**Решение**: Заполните колонку адресов данными

### 3. Только заголовки, нет данных
**Симптомы**: Ошибка "Нет данных для обработки"
**Решение**: Добавьте строки с данными в Excel файл

### 4. Адреса содержат только пробелы
**Симптомы**: Строки не определяются как заказы
**Решение**: Убедитесь, что адреса не пустые и не содержат только пробелы
